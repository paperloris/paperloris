# Paperloris

Single-page brand website for Paperloris — built with HTML5 and CSS3.

## Features

- **Coding font aesthetic** — JetBrains Mono throughout
- **Glassmorphism** — frosted glass cards, hero pane, and form
- **Dark/light mode** — toggle in header, `localStorage` persistence, system preference detection
- **Grid background** — subtle coding-grid pattern behind glass surfaces
- **Responsive** — mobile-first with hamburger nav, tablet, and desktop breakpoints
- **Accessible** — skip link, semantic HTML, ARIA, focus-visible, reduced-motion support

## Sections

1. Hero — glass pane with headline and CTAs
2. About — ethos narrative with code block
3. Services — three glass cards
4. Portfolio — six project image cards
5. Testimonials — three quote cards
6. Contact — glass form and studio info
7. Footer — navigation and copyright

## Open

Open `index.html` in any modern browser. No build step.
